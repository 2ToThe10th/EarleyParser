# EarleyParser

#### Запуск:

Для того чтобы запустить тесты,
воспользуйтесь командами:

```cmake -DDEBUG=1```

```make```

```./first-practical-work```

```cmake```

```make```

```./first-practical-work```

Для проверки программы на собственной кс-грамматике в main.cpp 
поменяйте вектор rules
